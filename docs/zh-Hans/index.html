技术支持

双视界（SceneFace）的技术支持、订阅帮助与法律信息。

双视界可在受支持的 iPhone 上同时录制前后镜头，并在启用语音识别时生成实时字幕， 最终导出带烧录字幕的竖屏视频。你可以在这里找到技术支持、隐私说明和使用条款。

联系支持

请发送邮件至 support@sceneface-app.newtree.tech。 为了更快定位问题，建议附上 iPhone 机型、iOS 版本、App 版本、复现步骤，以及问题是否和字幕、导出、权限或 SceneFace Pro 购买有关。

常见帮助主题

设备兼容性

双镜头录制依赖 AVCaptureMultiCamSession，双视界面向 iPhone XS 及更新机型。

权限说明

相机和麦克风权限为录制所必需。语音识别权限为可选，仅在你开启自动字幕时使用。

订阅与购买

SceneFace Pro 通过 Apple 计费。如果购买后没有解锁，请在 App 设置页中使用“恢复购买”。

导出行为

双视界导出的是竖屏视频。录制时前置预览为镜像显示，导出结果中的前置画面不镜像。

常见问题

为什么没有实时字幕？

请检查设置中是否开启“自动字幕”，并确认已授予“语音识别”权限。即使没有字幕，视频录制仍可继续。

为什么不能保存到照片？

只有在你主动保存导出视频时，双视界才会请求“仅添加到照片”权限。你也可以直接分享导出文件而不保存到照片。

如何管理或恢复 SceneFace Pro？

打开 App 内设置页，然后根据当前账户状态使用“管理订阅”“打开客户中心”或“恢复购买”。

提交 Bug 时需要附上什么？

建议提供设备型号、iOS 版本、App 版本、问题复现步骤、屏幕上的报错文案，以及问题是否稳定复现。

联系前建议先确认

  • 问题是否每次录制都会出现，还是只发生在单个素材上。
  • 相机、麦克风、语音识别或照片权限是否被拒绝。
  • 如果是购买问题，请说明 Apple ID 所在地区，以及是否已尝试“恢复购买”。
  • 如果是导出问题，请说明使用的是“性能优先”还是“灵活导出”模式。
