技術支援

雙視界（SceneFace）的技術支援、訂閱協助與法律資訊。

雙視界可在支援的 iPhone 上同時錄製前後鏡頭，並在啟用語音辨識時產生即時字幕， 最後匯出帶有燒錄字幕的直式影片。你可以在這裡找到技術支援、隱私說明與使用條款。

聯絡支援

請寄信至 support@sceneface-app.newtree.tech。 為了更快排查問題，建議附上 iPhone 機型、iOS 版本、App 版本、重現步驟，以及問題是否與字幕、匯出、權限或 SceneFace Pro 購買有關。

常見支援主題

裝置相容性

雙鏡頭錄製依賴 AVCaptureMultiCamSession，雙視界面向 iPhone XS 及更新機型。

權限說明

相機與麥克風權限為錄製所必需。語音辨識權限為可選，僅在你開啟自動字幕時使用。

訂閱與購買

SceneFace Pro 透過 Apple 計費。如果購買後沒有解鎖，請在 App 設定頁中使用「恢復購買」。

匯出行為

雙視界匯出的是直式影片。錄製時前置預覽為鏡像顯示，匯出結果中的前置畫面不鏡像。

常見問題

為什麼沒有即時字幕？

請確認設定中是否開啟「自動字幕」，並確認已授予「語音辨識」權限。即使沒有字幕，影片錄製仍可繼續。

為什麼不能儲存到照片？

只有在你主動儲存匯出影片時，雙視界才會要求「僅加入照片」權限。你也可以直接分享匯出檔案而不儲存到照片。

如何管理或恢復 SceneFace Pro？

打開 App 內設定頁，然後依照目前帳戶狀態使用「管理訂閱」「開啟客戶中心」或「恢復購買」。

回報 Bug 時要附上什麼？

建議提供裝置型號、iOS 版本、App 版本、重現步驟、畫面上的錯誤文字，以及問題是否可穩定重現。

聯絡前建議先確認

  • 問題是否每次錄製都會出現，還是只發生在單一素材上。
  • 相機、麥克風、語音辨識或照片權限是否被拒絕。
  • 若是購買問題，請說明 Apple ID 所在地區，以及是否已嘗試「恢復購買」。
  • 若是匯出問題，請說明使用的是「性能優先」還是「靈活匯出」模式。
